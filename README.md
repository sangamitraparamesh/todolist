# ✅ Simple To‑Do List

A lightweight, single-file **To‑Do List web app** built with **HTML, CSS, and JavaScript**, featuring local storage persistence, inline editing, filters, and drag-and-drop reordering.

---

## **Features**

* Add tasks with optional due dates
* Edit tasks inline by double-clicking
* Mark tasks as completed
* Delete tasks individually
* Clear all completed tasks
* Filter tasks: All / Active / Completed
* Drag and drop tasks to reorder
* Persistent storage using **localStorage**
* Keyboard shortcut: `Ctrl/Cmd + K` focuses the input field

---

## **Usage**

1. Open `todolist.html` in a web browser.
2. Type a task in the input box and press **Enter** or click **Add**.
3. Use the filters to view active or completed tasks.
4. Double-click a task to edit it inline.
5. Drag tasks to reorder them.
6. Click **Clear completed** to remove all finished tasks.

---

## **Technology Stack**

* **HTML** – Structure
* **CSS** – Styling, responsive design
* **JavaScript** – Functionality, localStorage, drag-and-drop
